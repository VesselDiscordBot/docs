## Description

Sync the database for a guild

## Arguments

| Name  | Description          | Type   | Required | Default Value                          |
| ----- | -------------------- | ------ | -------- | -------------------------------------- |
| guild | The guild id to sync | Option | No       | ID of the guild the command was ran in |
