## Description

Roll 1 or more die

## Arguments

| Name  | Description                              | Type   | Required | Default Value |
| ----- | ---------------------------------------- | ------ | -------- | ------------- |
| count | The amount of die to roll                | Number | No       | 1             |
| sides | The amount of sides each die should have | Number | No       | 6             |
